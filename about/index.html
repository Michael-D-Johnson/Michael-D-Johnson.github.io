The Data

Data for this visualization was taken from the American Housing Survey: https://www.huduser.gov/portal/datasets/hads/hads.html. This dataset includes a wide range of national housing affordability metrics starting from 1985 to 2013 covering odd years. The dataset includes information about cost of housing, burden of housing (housing cost/income), owning versus renting, and subsidized housing.

Conclusions are drawn from the two charts within this interactive slideshow: “Average Housing Costs: Subsidized vs. Unsubsidized” and "Average Household Salaries: Subsidized vs. Unsubsidized." For those on subsidized housing, household salaries dropped while housing costs remained constant. The fact that housing costs remained relatively constant for those on subsidized housing is expected, since subsidies are meant to control housing costs. What is interesting is that, since the recession, household salaries for those on subsidized housing have decreased and not improved. Conversely, for those not on subsidized housing, household salaries and household costs have increased at a similar rate, suggesting a smaller impact on housing affordability compared with those on subsidized housing. This demonstrates that a greater burden of housing costs lies with those on subsidized housing. 

What is most interesting is that salaries for those on subsidized housing were decreasing well before the 2008 recession. We are seeing the preconditions for the recession. That is, as my first assignment listed above showed, housing values were skyrocketing in the years prior to the recession. At the same time, as seen in this dashboard, household salaries were decreasing over the same period. This is consistent with the narrative that mortgage companies were selling homes in a market that was increasingly having difficulty affording such homes.

The Visualization

The visualization is an interactive slideshow. There are two data slides and one overview slide. On each slide (except the overview slide), the user can choose to interact and filter the data in a couple of ways. As the user progresses through the slideshow, narrative text is included to give overall context and move the user through the slideshow. The explanation of the data is summarized in the overview slide for those who want to bypass the narrative structure.

The progression through the slideshow is aided by text summaries below each chart in order to narrate important points along the way. This progression is facilited by a "Next" and "Back" button in the upper right of the scene. This narrative progression provides the basis for each "scene." Each scene is anchored by a chart, which is similar in size, scale, and aesthetic to the previous slide ensuring continuity. Similarly, the dropdown menu and navigation bars are also similarly located to help aid continuity between scenes. Lastly, the text provides the "story" or reason why one moves through the slides.

Each chart includes annotations that describe interesting data points. These points are put into context in the description below each chart. The annotations are transitioned into the scene in a sequence according to their Year (chronologically). This is done to aid the feeling of journeying through the narrative.

While on each scene the user can filter the data displayed by selecting parameters from a dropdown menu in the top left of the scene. The user can select from three parameters: All Data, Before 2000, and After 2000. When a parameter is selected, the chart changes to plot data only within the specified range. In essence, each parameter serves as a "zoom-in" of the data, giving the user a different view of certain time spans.

Furthermore, the user can interact with each data point (using d3 triggers). By hovering over a given data point, the user can get an easy-to-read printout of the data values.
